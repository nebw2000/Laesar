# Laser Show 

Interaktive Laser-Show Web-App.

## Features

- MP3, WAV, OGG Upload
- Zufällige Laser-Effekte basierend auf Beat & BPM
- 6 Grund-Effekte mit 75 Untervarianten
- Download der Animation als WebM Video
- Browserbasiert, läuft auf modernen Desktop-Browsern

## Nutzung

1. `index.html` auf GitHub Pages hosten oder lokal öffnen.
2. Musikdatei hochladen.
3. Animation wird automatisch gestartet.
4. Auf "Animation speichern" klicken, um Video zu exportieren.

## Hinweis

- iOS Safari: MediaRecorder evtl. eingeschränkt.
- Audio-Dateien müssen vom User hochgeladen werden.
